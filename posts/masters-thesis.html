Forecasting + computer vision frameworks to predict solid waste generation and estimate pile volumes.

Back Resume PDF

Abstract

Developed forecasting models for India by adapting lagged efficiency trajectories from other economies, and built a 2D image-based volume estimation system using depth maps in a controlled setup to estimate waste pile volumes at scale.

Contributions

  • Cross-country correlation + elasticity analysis of solid waste vs socioeconomic indicators; forecasting model for India.

  • 2D image-based volume estimation using depth maps (DINOv3) in a constrained geometry setup.

  • Millimeter-level height correction reported in the volume estimation pipeline.
